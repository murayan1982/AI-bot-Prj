# v2.3.0 Task List
# Public Facade Foundation

## Goal

v2.3.0 の目的は、外部アプリから AI Character Conversation Framework を呼び出すための
最小 public facade を用意すること。

デモアプリや外部アプリが `core/runtime.py`, `core/pipeline.py`, `llm/builder.py`,
`config/loader.py` などの内部構造に直接依存しないようにする。

v2.3.0 では、まず `text_chat` 相当の最小テキスト会話 API を対象にする。

---

## Policy

- `framework/` package を追加する
- `framework/facade.py` を public facade の入口にする
- 外部アプリは `framework.facade` を import する
- まずは text input -> LLM response -> text output を対象にする
- voice / STT / TTS / Live2D / VTS は本格対象にしない
- provider selection UX は v2.4 以降に回す
- OpenAI default 化はまだ行わない
- 既存の `main.py` 起動フローは壊さない
- facade は内部実装の薄いラッパーとして始める

---

## Public API Candidate

```python
from framework.facade import create_text_chat_session

session = create_text_chat_session(
    preset="text_chat",
    character_name="default",
)

response = session.ask("こんにちは")
print(response)
```

Primary API:

```text
create_text_chat_session()
TextChatSession.ask()
```

Optional / later API:

```text
TextChatSession.ask_stream()
TextChatSession.reset()
```
